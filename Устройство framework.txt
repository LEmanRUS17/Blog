Структура базовой сборки yii 2

assets/     - содержит подключения ресурсов (js, css)
commands/   - содержит консольные команды
components/ - содержит виджеты
libs/       - содержит функции для разработки

config/ - содержит конфигурационные файлы
    console.php - конфигурация консольного приложения
    db.php      - конфигурация подключения к базе данных
    params.php  - дополнительные параметры
    web.php     - конфигурация веб приложения

controllers/ - содержит классы контроллеров
    SiteController.php - контроллер главной страницы и страницы 404
    HomeController.php - контроллер для работы с домашней страницей
    AppController.php  - общий контроллер от которого наследуются все остальные контроллеры

mail/     - содержит файлы представлений для электронной почты
migration/ - содержит файлы миграции в БД
models/   - содержит классы моделей
modules/  - содержит модули
    admin/ - содержит компоненты для работы в админке

runtime/  - содержит файлы, созданные во время работы сайта (логи, кэш)
tests/    - содержит тесты для основных приложений
vendor/   - содержит фреймворк Yii и сторонние библиотеки

views/ - содержит файлы вида
    auth/ - содержит страницы для работы с пользователями
        login.php  - страница входа пользователя в акаунт
        signup.php - страница регестрации
    home/ - содержит страницы сайта
        category.php - страница по категориям
        index.php    - главная страница вывода статей
        single.php   - вывод статьи
    layouts/ - содержит шаблоны
        blog.php  - оснавной шаблон сайта
    partials/ - содержит вспомогательные элементы
        comment.php - вывод коментариев
        sidebar.php - список статей и категорий
    site/ - содержит отдельные файлы страниц
        about.php
        contact.php
        error.php
        index.php
        login.php
web/ - основная папка сайта, содержит файлы стилей, скрипты, картинки и т.д.
    css/           - содержит файлы стилей 
    images/        - содержит картинки
    js/            - содержит файлы скриптов
    favicon.ico
    index.php      - точка входа
    index-test.php
    robots.txt
